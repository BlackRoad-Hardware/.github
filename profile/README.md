## About

BlackRoad Hardware manages the physical edge of the BlackRoad fleet -- Raspberry Pis, Hailo-8 accelerators, sensors, and IoT devices. We write the firmware, build the dashboards, and orchestrate fleets of edge nodes that run AI inference, mesh networking, and environmental monitoring at the hardware level.
